h1. ax_ipcheck

p. This plugin allows for restricted assess to "Textpattern":https://textpattern.com/ websites, pages or sectiions of code by checking for an allowed IP range of the client.

p. REQUIRES: "Textpattern CMS":https://textpattern.com/, tested with v4.7.2 and v4.8.4

p. "DOWNLOAD":https://raw.github.com/eliph/ax_ipcheck/master/ax_ipcheck.txt

h2. Tags

bc. <txp:ax_ipcheck fromip="192.168.0.0" toip="192.168.255.255">
    IP ok
<txp:else />
    IP not ok
</txp:ax_ipcheck>
